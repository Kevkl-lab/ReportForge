# PenForge

> Modern penetration testing report manager written in C++ and Qt.

PenForge is a desktop application designed for documenting penetration tests and generating professional PDF reports.

The goal of this project is to provide a lightweight, modern alternative for individual penetration testers, students and cybersecurity enthusiasts.

---

## Features

- 📁 Project management
- 🛡 Finding management
- 📊 Severity & CVSS support
- 📄 Professional PDF report generation
- 🗂 SQLite database
- 🌙 Modern dark UI
- 📷 Screenshot support (planned)
- 📑 Templates (planned)

---

## Tech Stack

- C++23
- Qt 6
- CMake
- SQLite
